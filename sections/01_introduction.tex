\chapter{Introduction}

\section{Contexte}
Le présent rapport documente l'audit de sécurité de l'application web OWASP Juice Shop, une plateforme intentionnellement vulnérable conçue pour l'apprentissage et la formation en sécurité des applications web.

\section{Objectifs de l'audit}
Les objectifs de cet audit sont :
\begin{itemize}
    \item Identifier les vulnérabilités présentes dans l'application
    \item Démontrer les techniques d'exploitation
    \item Proposer des recommandations de correction
    \item Documenter les bonnes pratiques de sécurité
\end{itemize}

\section{Application choisie}
OWASP Juice Shop est une application web moderne développée en Node.js avec le framework Express. Elle présente une large gamme de vulnérabilités courantes, classées par niveau de difficulté (de 1 à 6 étoiles).

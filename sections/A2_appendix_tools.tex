\chapter{Outils utilisés}

\section{Burp Suite}
Burp Suite est un outil de test de sécurité web permettant d'intercepter, de modifier et d'analyser les requêtes HTTP.

\section{Navigateur web}
Firefox et Chrome ont été utilisés pour naviguer sur l'application et observer le comportement du frontend.

\section{Scripts Python}
Des scripts Python ont été écrits pour automatiser certaines tâches d'exploitation.

\section{Terminal}
Le terminal a été utilisé pour lancer l'application et exécuter des commandes.

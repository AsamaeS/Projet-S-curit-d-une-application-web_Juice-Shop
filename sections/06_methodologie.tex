\chapter{Méthodologie d'audit}

\section{Outils utilisés}
Les outils suivants ont été utilisés pour réaliser cet audit :
\begin{itemize}
    \item Navigateur web (Firefox/Chrome)
    \item Burp Suite (interception, modification de requêtes)
    \item Terminal/CLI
    \item Scripts Python
\end{itemize}

\section{Environnement de test}
L'application a été testée dans un environnement local :
\begin{itemize}
    \item Système d'exploitation : Windows/Linux
    \item OWASP Juice Shop : version locale
    \item Réseau : localhost (isolation du réseau)
\end{itemize}

\section{Approche de test}
L'audit a suivi une approche méthodique :
\begin{enumerate}
    \item Reconnaissance de l'application
    \item Identification des points d'entrée
    \item Test des vulnérabilités par catégorie
    \item Documentation des découvertes
    \item Validation des exploits
\end{enumerate}
